# todo List

### 说明

使用工具 `react` + `javascript` + `vite`

使用 `vite` 创建一个react项目，比react脚手架 `create-react-app`创建更快，可选方案更灵活；

`vite` 创建命令 `npm create vite@latest`；

### 项目说明

+ 该项目主要采用类式组件；

+ 父子组件的通信方式，采用状态升级，即将通用数据放到`App`父组件的 `state` 中；
