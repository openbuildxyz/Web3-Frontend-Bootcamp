## [分析题] 比较区块链与传统数据库，你的看法？

```
区块链：适合小数据量存储，公开数据存储，去中心化，人手一份数据库内容，一旦上链不可被篡改
传统数据库：大容量存储，私有，中心化管理，容易被盗取篡改
```
