## [简答题] 简述区块链的网络结构

```
区块链分为数据层、网络层、共识层、应用层。数据层包含交易区快，网络层负责节点之间的通信，共识层负责验证区块并排序，应用层构建各种各样的应用。
```

## [简答题] 智能合约是什么，有何作用？

```
智能合约是部署在区块链上的可执行程序，负责处理区块链上的请求，执行智能合约的逻辑
```

## [简答题] 怎么理解大家常说的 `EVM` 这个词汇？

```
EVM是以太坊虚拟机，用来执行智能合约
```
