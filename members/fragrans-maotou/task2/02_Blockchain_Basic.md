## [简答题] 简述区块链的网络结构

```
是一种链式结构，后一个区块中包含上一个区块的hash值，保证整个区块链是安全的。
```

## [简答题] 智能合约是什么，有何作用？

```
问：智能合约是什么？
答：智能合约类似于在web2服务器上可运行的程序，只不过智能合约运行在区块链上。

问：有何作用？
答：智能合约是用户之间在链上进行安全交易的程序。
```

## [分析题] 你对去中心化的理解

```
用户的数据不在集中的存储在某一个机构里，用户的数据在用户自己的手中（私钥），用户交易的数据，在区块链上不集中在单个机构手中。
```

## [分析题] 比较区块链与传统数据库，你的看法？

```
区块链上的数据透明，且不能轻易修改。
```
