# SnowDex

## Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
# or (Recommend)
bun install
```

## Development

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or (Recommend)
bun dev
```
