# Rare Disease Charity DAO

罕见病慈善 DAO

## 项目概述

一个支持多链的，为罕见病患者及其家庭，而发行代币 / NFT 进行募资的慈善 DAO 平台。

## 技术架构

**前端：**

-   React + TypeScript + Vite + Rainbowkit + Wagmi

**后端：**

-   Foundry + Solidity

## 核心功能

整个项目分为 **2 个部分**：

**1. DAO 审核员部分**

根据登录身份的不同，可执行的操作如下：

-   任何人

    -   可以查看 DAO 审核员名单及其信息；

-   非审核员

    -   可以申请新审核员身份；

-   审核员
    -   可以修改自身相关信息；
    -   可以批准 / 驳回新申请;
    -   可以质疑现有审核员（提交不信任提案）；
    -   可以赞成 / 反对不信任提案；
    -   可以退出审核员身份；<br><br>

**2. 慈善项目部分**

根据登录身份的不同，可执行的操作如下：

-   任何人

    -   可以查看已发行项目及其信息；
    -   可以申请发行新代币 / NFT 慈善项目；

-   审核员

    -   可以批准 / 驳回新项目申请;

-   被授权的项目发起人

    -   可以发行新代币 / NFT 慈善项目。
